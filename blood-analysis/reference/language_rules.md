# 言葉づかいルール — 断定しない、予測する

このSkillの核となる姿勢。出力前に必ずチェックする。

## 使わない言葉

| ❌ NG | ✅ OK |
|---|---|
| 〜である | 〜と読み取れます／〜のように見えます |
| 〜が原因です | 〜が一因の可能性があります |
| 〜と診断されます | 〜と推測されます／〜という仮説が立ちます |
| 〜が起きています | 〜が起きている可能性があります／〜のサインかもしれません |
| 確実に〜 | 〜の確度は高い／〜が強く示唆されます |
| 必ず〜してください | 〜を試してみる価値がありそうです |
| 異常です／正常です | 理想値より乖離があります／理想値の範囲内です |

## 推奨フレーズ

- 「〜の可能性があります」
- 「〜のサインとして読み取れます」
- 「〜が予測されます」
- 「〜という仮説が立てられます」
- 「〜という流れで進んでいる可能性があります」
- 「次の検査・問診で確かめたい点です」
- 「現時点での見立てとしては〜」
- 「数値からは〜と読めますが、確定には追加情報が必要です」

## スコア・判定の表現

- スコアは「**現時点での仮の見立て**」と明記。
- レーダーチャート・ゲージ・信号機は「観測の方向性を視覚化したもの」であり、診断ではない。
- 「Critical」「Watch」「Stable」のような英語ラベルも、レポート内で「現時点で注目しておきたい」「経過を見たい」「現状は良好」というニュアンスに翻訳する。

## CORE PATTERN の書き方

良い: 「糖代謝の劣化を起点に、糖化＋酸化が進行している**可能性**が見えます」
悪い: 「糖代謝が悪化し、酸化ストレスが進行しています」

良い: 「これらの3つは別の問題ではなく、ひとつながりの現象として進んでいる**可能性**があります」
悪い: 「これらの3つは連動して進行しています」

## 「次の一歩」の書き方

良い: 「塩分を6g/日にしてみる価値がありそうです」「3週間試してみると、舌のリセットが感じられるかもしれません」
悪い: 「塩分を6gにしてください」「3週間で効果が出ます」

## バイリンガル表記の方針（PDF/UI ラベル）

レポートの大見出しは「英語タイトル + 日本語サブタイトル」の二段組がブランド設計だが、**機能ラベル・メタ情報・補助テキストは日本語を主役にする**。

### 日本語を使うべき箇所（英語のみは避ける）

| 箇所 | ❌ NG | ✅ OK |
|---|---|---|
| カバーメタ | `Subject / Male 45` | `対象 / 小林大介 男性 45歳` |
| カバーメタ | `Date / 2025.10.26` | `解析日 / 2025.10.26` |
| ピル/バッジ | `Critical` / `Watch` / `Stable` | `注目` / `経過観察` / `安定` |
| ピル/バッジ | `Improving` | `改善傾向` |
| Stepカード | `Nutrition` / `Daily Habit` / `Follow-up` | `食事` / `毎日の習慣` / `次の検査` |
| 列ヘッダ | `Nutrition` (大) + `食事` (小) | `食事` (大) + `Nutrition` (小) |
| ゲージ/グラフラベル | `Overall Score` | `総合スコア` |
| ゲージ/グラフラベル | `System Map` | `体のシステム地図` |
| カードタグ | `CORE PATTERN` (単独) | `中心テーマ` + 補助で `CORE PATTERN` |

### 英語をそのまま使ってよい箇所
- セクション大見出しの英語タイトル（例: `Summary`、`Findings`、`Your Story`）— **必ず日本語サブタイトルとペアで**
- ブランド名（`MitoFlow40`、`Vol. 001` 等）
- グラフ内のテクニカルラベル（`HRV`、`HbA1c` 等の医学略語）

### 原則
- **読み手は日本語話者**。意味の通る日本語があるのに英語にする理由はない
- 英語は「デザイン上のアクセント」「医学用語」に限定
- バイリンガル並列は英語が主役ではなく、日本語の補強として置く

## 全体の姿勢

- 解析は **「答え」ではなく「地図」**
- レポートは **「結論」ではなく「次の検査・問診への問いかけ」**
- 数値は **「現在地」ではなく「動きの方向の予測」**
- 重要なのは **「断定の自信」ではなく「複数の可能性を開いたまま提示する誠実さ」**
