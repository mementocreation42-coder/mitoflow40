### 3. CORE PATTERN（中心テーマ）

> 個別所見の羅列ではなく、**「何が起点で、何に波及しているか」**を1〜2文で明示する。

例: 「糖代謝の劣化を起点に、酸化ストレス対応の弱化（γ-GT低）と栄養補給の不足（Hb/RBC低下）が連動している」

**起点 → 波及のフロー図（3〜4ノード）**:

```
01 [起点となる系] → 02 [連動する系] → 03 [結果として現れる系]
   主な所見             主な所見            主な所見
```
